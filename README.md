# Cloud Ops Helper

A practical skill for cloud infrastructure — deployment, monitoring, CI/CD, secrets management, and automation.

## Installation

- **Cursor:** Place `SKILL.md` in `.cursor/skills/cloud-ops-helper/SKILL.md`
- **Claude Code:** Add `SKILL.md` to `.claude/skills/` directory

## What it does

Guides AI agents to write better cloud ops code by:
- Suggesting appropriate packages for cloud provider interactions
- Providing patterns for monitoring and observability
- Including CI/CD pipeline best practices
